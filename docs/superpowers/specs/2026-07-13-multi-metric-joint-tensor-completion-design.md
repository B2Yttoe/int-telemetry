# 多指标联合张量补全设计

## 目标

在完全相同的 INT probe、hop/report、Ground OAM 输入、直接观测掩码和遥测字节下，对比逐指标二维低秩补全与多指标联合张量补全，验证联合建模是否降低节点/链路各指标误差并减少跨指标物理一致性违反。

## 张量与掩码

节点张量为：

\[
\mathcal X^V\in\mathbb R^{T\times N\times M_V},\quad
M_V=[CPU,Queue,QueuedTraffic,Cache,Energy].
\]

链路张量为：

\[
\mathcal X^E\in\mathbb R^{T\times L\times M_E},
\]

其中链路指标包括利用率、传播/排队时延、容量、拥塞、排队/丢弃流量和误包率。链路目录保持稳定，动态可用性由 active mask 表示；`topology-down` 不是缺失值，禁止补全。

每个张量同时维护：

- `observed_mask[t,o,m]`：该指标是否由已送达 INT 报告直接观测；
- `active_mask[t,o,m]`：该对象-指标单元在物理拓扑中是否有效；
- `prior[t,o,m]`：现有时间、轨道邻居和空间分组先验初始化。

直接观测值在训练后重新锁定，所有归一化统计只从 observed mask 内计算。

## 联合模型

第一版采用确定性 CP 分解：

\[
\hat x_{t,o,m}=\mu_m+sigma_m
\sum_{r=1}^{R}a_{t,r}b_{o,r}c_{m,r}.
\]

时间因子、对象因子和指标因子由所有已观测指标共同训练。缺失值使用联合预测与结构先验的保守混合：

\[
\hat x^{final}_{t,o,m}
=\rho\hat x^{CP}_{t,o,m}+(1-\rho)x^{prior}_{t,o,m}.
\]

对象因子沿 Walker 轨道邻居做轻量平滑，时间因子沿相邻时间片平滑；这两个正则项不读取隐藏目标值。

## 两个联合后端

- `joint-cp`：只执行联合 CP 张量补全；
- `joint-cp-physics`：在 `joint-cp` 后，对 inferred 单元启用现有无真值、保守门控的节点工作负载/能耗一致性和链路利用率/队列/拥塞/损失一致性投影。

物理投影不修改直接观测值，不使用补全误差或第一阶段隐藏指标值。

## 注意事项

1. 百分比、毫秒、MB 和误包率按指标分别归一化，统计量只来自直接观测。
2. `mode/status` 使用派生分类与 F1/accuracy 评价，不作为连续张量通道。
3. 链路 active mask 独立于 observed mask；物理 down 单元保持 down。
4. 第一版属于离线重构，不声称未来预测；相邻时间片观测可用于补全。
5. 同一实验内三种方法共享输入 SHA-256、观测掩码、遥测字节和能耗。
6. 不用隐藏真值选择混合权重或正则参数；固定参数在实验前声明。
7. 同时报告每指标 MAE/RMSE/P95/R2、宏平均归一化误差、物理一致性违反率、墙钟时间和内存代理。

## 第一版实验

比较：

1. `low-rank`：每个指标独立二维补全；
2. `joint-cp`：多指标联合张量补全；
3. `joint-cp-physics`：联合张量补全加保守物理投影。

主要判据不是要求联合方法所有指标必然获胜，而是检查：在同一观测和开销下，宏平均误差是否下降、至少多数关键指标是否改善、物理一致性违反率是否下降，以及额外 Ground OAM 计算成本是否可接受。
